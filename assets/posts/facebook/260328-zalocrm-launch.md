### 🚀 Cài đặt chỉ 3 phút

```
git clone https://github.com/locphamnguyen/ZaloCRM.git
cd ZaloCRM && cp .env.example .env
docker compose up -d --build
```

Quét QR Zalo → Bắt đầu nhận tin nhắn.

- Phân quyền — ai xem gì, ai chat gì
